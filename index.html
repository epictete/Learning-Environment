HTML : Balises de base

Les niveaux de titres

En HTML on peut utiliser jusqu'à 6 niveaux de titres avec les balises <h>

<h1> sera un titre de niveau 1.

<h2> sera un titre de niveau 2.

<h3> sera un titre de niveau 3.

Etc. jusque <h6> qui sera un titre de niveau 6.

Le formatage de texte

Le HTML permet de formater du texte avec une syntaxe simple.

Italique

Pour mettre du texte en italique il y a une méthode :

<i>exemple</i> donne exemple.

Gras

La manière de mettre le texte en gras est similaire à l'italique. Il existe deux méthodes:

<b>exemple</b> donne exemple.

<strong>exemple</strong> donne exemple.

Barré

Pour barrer un mot ou une phrase, il suffit de l'encapsuler comme ceci <del>exemple</del>.

Il existe une autre manière pour y arriver : <s>exemple</s>.

Les listes

Le HTML nous permet de faire des listes, ordonnées ou non, avec la possibilité de les imbriquer.

Liste à puce

Pour réaliser une liste à puce, on peut utiliser les balises <ul> pour créer la liste, avec <li> pour les éléments.


<ul>
<li>Par
<li>exemple
    <ul>
    <li>omme
    <li>ceci
    </ul>
<li>!
</ul>

Donne:

  • Par
  • Exemple
    • Comme
    • Ceci
  • !

Liste ordonnée

Pour la liste ordonnée, c'est la meme syntaxe que pour la liste à puce sauf qu'on remplace les symboles <ul> par <ol>.


<ol>
<li>Par
<li>exemple
    <ol>
    <li>omme
    <li>ceci
    </ol>
<li>!
</ol>

Donne:

  1. Par
  2. Exemple
    1. Comme
    2. Ceci
  3. !

Insertion d’une image statique

Les images s'insèrent en utilisant la balise suivante <img src="" alt="">.

Le lien de l'image setrouve entre les guillements src.

Le texte alt entre guillemets est le texte alternatif de l'image.

Les tables

Pour créer un tableau, il faut d'abord créer les balises : <table> et </table>

Il faut ensuite créer des lignes avec les balises : <tr> et </tr>

Et enfin ajouter les cellules en utilisant les balises : <td> et </td>

Il existe un cas particulier de cellule pour les en-têtes. Dans ce cas il fuat utiliser les balises : <th> et </th>

Voici un exemple de code :


<table>
    <tr>
        <th>En-tête 1</th>
        <th>En-tête 1</th>
    </tr>
    <tr>
        <td>Contenu 1</td>
        <td>Contenu 2</td>
    </tr>
    <tr>
        <td>Contenu 3</td>
        <td>Contenu 4</td>
    </tr>
</table>

Résultat :

En-tête 1 En-tête 1
Contenu 1 Contenu 2
Contenu 3 Contenu 4
